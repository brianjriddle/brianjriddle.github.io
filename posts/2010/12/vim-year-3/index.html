This is the start of my third year with vim. Before that I used emacs for about 7 years almost daily. A couple things made me switch

  • I was using windows and I upgraded x-emacs one day and pretty much botch development environment
  • I realized that I was using vi/vim everyday because I was Permanently connected to some server via ssh and it was time to get more efficient.
  • I really needed to follow the use one editor and use it well from the pragmatic programmer.

I use vim almost exclusively today with one glaring exception. java. For that I use Intellij as it has the best refactoring support of any of the java editors I have used. Don’t know if I will ever rectify that but as I am writing less and less java these days it’s not anything I think about much.

My use of vim really took off after reading Tim Ottinger’s use vim like a pro. If you are starting of with vim today I would still recommend using that but think that Yehuda katz blog post Everyone Who Tried to Convince Me to use Vim was Wrong is probably a better place to start if you want a practical getting started now. Then there is vimcasts if you are wanting a good introduction in how to use vim.
